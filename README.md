# 小彩金两周年纪念网页

这是为小彩金制作的两周年纪念网页，包含时间轴回忆、照片轮播、视频播放和背景音乐。

## 在线访问

部署后可通过以下链接访问：
https://yuancao1996.github.io/anniversary

## 文件说明

- `index.html` - 主网页文件
- `三周年素材/` - 存放照片和视频素材

## 背景音乐

A Thousand Years - Christina Perri

## 技术特性

- 响应式设计，支持手机和电脑
- 自动播放背景音乐
- 烟花动画效果
- 时间轴展示
- 照片和视频展示
